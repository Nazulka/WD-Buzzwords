Welcome tp the WD Buzzwords Glossary

Web technologies contain long lists of jargon and abbreviations that are used in documentation and coding. From Java to JavaScript, frontend to backend and apps to APIs, it’s normal to feel overwhelmed by the seemingly endless stream of new terminology. This glossary provides definitions of words and abbreviations you need to know to successfully understand and build for the web.
